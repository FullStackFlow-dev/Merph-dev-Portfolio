## ⚙️ Comment ça marche (CI/CD)

1. Tu pousses un commit sur `main`
2. GitHub Actions déclenche le pipeline automatiquement
3. **Validation** — vérifie que tous les fichiers requis sont présents
4. **Link Check** — vérifie que tous les liens externes fonctionnent
5. **Report** — génère un rapport de progression automatiquement
6. **Deploy** — pousse la documentation sur GitHub Pages
